This GuitarTuner applet implements a Fast Fourier Transform to find fundamental frequency. You perhabs need to install Java plugin v1.5 to run it correctly.

Accordez facilement et précisement votre guitare en ligne, en jouant devant votre microphone. Ce logiciel libre et gratuit est un applet Java qui fonctionne comme un accordeur réel !
